Khuyến nghị:
- Ảnh dọc hoặc ảnh có nhân vật ở chính giữa.
- Kích thước tối thiểu 1200 x 1600 px.
- Mỗi ảnh nên dưới 1.5 MB để tải nhanh trên điện thoại.
- Nếu dùng tên hoặc đuôi file khác, sửa mảng "slides" gần cuối index.html.
